# CrisisLiveTxt-Dataset - A Large-Scaled Crisis-related Dataset for Humanitarian Information Detection on Social Media

Detection and classify Event crisis information in social media is crutial for Humanitarian Operation. This dataset simulates real world scenarios of the task by:

- Gathering and selecting 158,975 records from 14 previous crisis-related annotated datasets.
- Mix with another 11 noise datasets from Twitter which contain close topic to the crisis-related dataset.


We temporary cannot public the dataset due to we under the publishcation process and [X Developer Agreement](https://docs.x.com/developer-terms/agreement).
